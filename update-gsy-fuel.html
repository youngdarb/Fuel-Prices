GitHub Token

A GitHub token lets the admin panel publish directly to the live site. You only need to do this once — it's saved in your browser.

Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained, create a token with Contents: Read and write for the Fuel-Prices repo.

All done? Hit Publish.
Changes go live on the website in about 30 seconds.
